# 1111.tn — Homepage (Next.js)

A professional, pixel-faithful clone of the 1111.tn price-intelligence homepage, built with **Next.js 14 (App Router)**, **TypeScript**, **Tailwind CSS**, and **Recharts** for animated charts.

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## What's inside

- Sticky header with bilingual (FR / AR) navigation
- Hero with animated mascot, search and real-time market index card (animated sparkline)
- TOP OFFRES horizontal carousel with real product images (Unsplash)
- Category pills (FR + AR labels)
- "Grande distribution" panier comparison, Veille prix, Alerte prix, Prévision IA
- Stat row: dernier article modifié, dernier ajouté, total SKU, promos illogiques
- Qoffa Tounsi (panier familial) with weekly chart
- Baromètre Climatiseurs with multi-line price chart and AI alert
- IA prédictive section with historical + forecast lines
- Tunisia E-commerce Market Index (4 sub-indices + meta index)
- Baromètres par catégorie
- Observatoire + Ooredoo banner
- App download banner (App Store / Google Play)
- Footer with social links and rating

## Stack

- Next.js 14, React 18, TypeScript
- Tailwind CSS (custom dark theme + brand palette)
- Recharts (animated area + multi-line charts)
- Lucide-react icons
- Inter + Tajawal (Arabic) fonts
